# How to use the brand.json file?

The `brand.json` file is used to configure the links and brand on your documentation. It is located in the root of your project.

1. Use "name" field to set the name of your project. It is reflected at the top left corner of the documentation.
2. Use "login_url" field to set the login url of your project. It is reflected at the top right corner of the documentation.
3. Use "signup_url" field to set the signup url of your project. It is reflected at the top right corner of the documentation.
