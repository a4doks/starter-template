# Styling

Enhance your documentation by leveraging these versatile components.

## Accordion
<Separator/>

Perfect for FAQs or any content you wish to condense.

```markdown
<Accordion type="single" collapsible>
  <AccordionItem value="item-1">
    <AccordionTrigger>What is A4Doks?</AccordionTrigger>
    <AccordionContent>
      A4Doks is a powerful documentation platform that transforms markdown files into beautiful documentation sites.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-2">
    <AccordionTrigger>How do I sign up?</AccordionTrigger>
    <AccordionContent>
      Start by integrating your GitHub account and fork our boilerplate repository.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-3">
    <AccordionTrigger>Are all Shadcn components supported?</AccordionTrigger>
    <AccordionContent>
      As of today, only the ones you see here are supported. We are working hard to support all Shadcn components.
    </AccordionContent>
  </AccordionItem>
</Accordion>
```

**Result:**

## Card
<Separator/>

Present content in a structured and visually appealing manner.

```markdown
<CardGroup>
    <Card title="Welcome" href="/welcome">
        Simplify documentation with powerful markdown and shadcn components.
    </Card>
    <Card title="Styling" href="/styling">
        Use shadcn components to enhance your documentation.
    </Card>
        <Card title="Structure" href="/page/structure">
        Learn how A4Doks structures your documentation.
    </Card>
    <Card title="Navigation" href="/page/navigation">
        Learn how to organize your left panel navigation.
    </Card>
</CardGroup>
```

**Result:**

    <Card title="Welcome" href="/welcome">
        Simplify documentation with powerful markdown and shadcn components.

## Checkbox
<Separator/>

Engage your readers with interactive checklists.

```markdown
<Checkbox id="minimal" label="Minimal" checked={true} />
<Checkbox id="beautiful" label="Beautiful" checked={true} />
<Checkbox id="easysetup" label="Easy to setup" checked={true} />
```

**Result**:

<Checkbox id="minimal" label="Minimal" checked={true} />
<Checkbox id="beautiful" label="Beautiful" checked={true} />
<Checkbox id="easysetup" label="Easy to setup" checked={true} />
